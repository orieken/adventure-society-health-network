# ASHN Presentations

Presentation-ready Markdown decks for project demos and stakeholder walkthroughs.

## Decks

- [ASHN Elevator Pitch Presentation](ashn-elevator-pitch.md) — 3–5 minute project pitch and demo narrative.
- [ASHN Technical Deep Dive Presentation](ashn-technical-deep-dive.md) — 20–30 minute architecture and implementation walkthrough.

## Suggested Use

- Start with the elevator pitch for executives, product partners, or first-time audiences.
- Use the technical deep dive for engineering reviews, architecture walkthroughs, or project planning.
- Pair either deck with the live dashboard at `http://localhost:9300` after running `make dev-stack`.
